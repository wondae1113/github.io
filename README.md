# 김민수 | 자기소개 페이지

소프트웨어 개발자 **김민수**의 자기소개 포트폴리오 웹사이트입니다.  
HTML, CSS, JavaScript만 사용하여 제작했습니다.

## 🌐 라이브 사이트

[https://wondae1113.github.io/](https://wondae1113.github.io/)

## 📁 프로젝트 구조

```
github.io/
├── index.html    # 페이지 구조 (HTML)
├── style.css     # 디자인 (CSS)
├── script.js     # 동작 (JavaScript)
├── P.png         # 프로필 사진
└── README.md     # 프로젝트 설명 (이 파일)
```

## 🛠 사용 기술

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📄 페이지 구성

| 섹션 | 내용 |
|------|------|
| 지원 동기 및 포부 | 개발 경험과 지원 동기 |
